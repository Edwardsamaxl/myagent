## Phase 2: 路由准确性测试

### 测试配置
- 模型：qwen2.5:7b（本地 Ollama）
- 问题集：30 条问题，6 个类别

### 测试结果

| 类别 | 准确率 |
|------|--------|
| 独立问题 | 5/5 = 100% |
| 多跳推理 | 5/5 = 100% |
| 指代消解 | 4/5 = 80% |
| 隐含意图 | 3/5 = 60% |
| 模糊表述 | 4/5 = 80% |
| 复合意图 | 5/5 = 100% |
| **整体** | **26/30 = 86.7%** |
